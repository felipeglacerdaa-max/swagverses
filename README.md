# swagverses

Projeto de loja `SwagVerse's` com layout streetwear / swag moderno, desenvolvido com Vite.

## Como usar

1. Rode `npm install` na pasta do projeto.
2. Use `npm run dev` para abrir o servidor de desenvolvimento.
3. Use `npm run build` para gerar a versão de produção.

## Arquivos principais

- `index.html` — entrada do Vite
- `src/main.js` — lógica do carrinho e navegação
- `src/style.css` — estilização moderna
- `package.json` — scripts e dependências
